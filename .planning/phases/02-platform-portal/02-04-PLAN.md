| Boundary | Description |
|----------|-------------|
| Client → URL State | nuqs manages pagination/filter params in URL |
| URL → Hook → API | Container hook translates URL state to API query params |
| API Response → DataTable | TanStack Query data to table rows |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-13 | Tampering | nuqs URL params | mitigate | useQueryStates enforces type parsing (parseAsInteger rejects non-integers) |
| T-02-14 | Information Disclosure | nuqs URL state | accept | Only filter/pagination params in URL (no sensitive data) |
| T-02-20 | Injection | DataTable column rendering | mitigate | Column cell renderers use React JSX (auto-escaped), no raw HTML injection |

</threat_model>
